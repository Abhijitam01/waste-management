# OceanCleanup Connect

A modern NGO platform for ocean waste detection and cleanup coordination.

## Quick Start

### 1. Start ML Service
```bash
cd ..
source venv/bin/activate
python app.py
```

### 2. Start Frontend
```bash
npm run dev
```

### 3. Open Browser
Navigate to http://localhost:3000
